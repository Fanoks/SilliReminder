# Code of Conduct

This project aims to be a welcoming, harassment-free space for everyone.

## Expected behavior
- Be respectful and professional.
- Assume good intent; ask clarifying questions.
- Keep feedback constructive and focused on the work.
- Respect user privacy and sensitive data.

## Unacceptable behavior
- Harassment, threats, or hate speech.
- Personal attacks or insulting/derogatory comments.
- Publishing private information without permission.
- Deliberate disruption of discussions (spam, trolling).

## Scope
This applies to all project spaces: issues, pull requests, discussions, and other community channels.

## Enforcement
Project maintainers may remove, edit, or reject content that violates this policy and may block users from participation.

## Reporting
If something happens that violates this Code of Conduct, please open an issue with minimal details **or** contact the maintainer privately.
